This example app requires a uaa service instance.

Use the XS-A CLI to create one with the following command.

xs create-service xsuaa default mta-python-uaa

Note: For the python links to work, you must git clone this project to a folder on the server and run tools/build.sh .

python/hello Provided by the Python module but doesn't check JWT.
spython/auth Provided by the Python module requires authorization but doesn't check JWT.
spython/valid Provided by the Python module requires authorization and checks JWT.

https://uaa-server.hana2sp03.sap-a-team.com/uaa-security/config?action=who Once authenticated, check session.
