This example app requires a uaa service instance.

Use the XS-A CLI to create one with the following command.

xs create-service xsuaa default python-uaa

Note: For the python links to work, you must git clone this project to a folder on the server and run tools/build.sh .

python/test Provided by the Python module but doesn't check JWT or access data.
python/test1 Provided by the Pyton module and accesses data.
spython/test2 Provided by the Pyton module and checks the JWT and accesses data.
